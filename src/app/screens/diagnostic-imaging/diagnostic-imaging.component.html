Equipamiento de última generación para realizar Tomografías de diagnótico.

El Centro cuenta con un tomógrafo SOMATOM go.Now para Diagnóstico con Imágenes de forma simple y efectiva.

Sus múltiples receptores ofrecen una mejor resolución y definen mejor el contorno de los órganos. Brinda cortes submilimétricos en todo el ancho del detector lo que permite imágenes ideales para tratamientos de radiocirugía y radioterapia, así como también para escaneos de rutina, angiografías, exámenes neurológicos y de estructuras óseas hasta las más pequeñas como las del oído interno.

La Braquiterapia HDR se indica para:

Brinda imágenes de alta calidad con cortes multislice.

Cuenta con CARE i-Tilt para la protección de órganos sensibles.

Modula el amperaje y voltaje para cada ancho de corte, lo que significa menor dosis para el paciente.
